üåê Infrastructure

The application infrastructure is based on the FARM stack, including FastAPI, React and MongoDB. She uses MongoDB for efficient data management, FastAPI to process queries, and React to deliver a user interface for viewing and interacting with data. The API and web client are independent. The API could be used by other clients, such as a mobile application.

FastAPI is used as a backend framework to handle HTTP requests and processing-related operations. database. It offers high performance thanks to automatic generation of documentation, Fast data validation and support for OpenAPI specifications.

MongoDB is the database chosen to efficiently store and manage application data. Its flexible JSON document-based data model makes it easy to store and retrieve data. information. Read/write operations are optimized for high performance.

React is used to create the dynamic user interface of the application. It allows to build reusable components and offers efficient rendering of views based on state management, ensuring a smooth and responsive user experience. We use it coupled with Tailwind CSS, a open-source CSS framework, to facilitate team user interface development.

The application is hosted on Render, a cloud computing platform. It is accessible to the next address: https://cafesansfil.onrender.com/.
