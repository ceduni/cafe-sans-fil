Infrastructure

The application infrastructure is based on the FARM stack, comprising FastAPI, React and MongoDB. It uses MongoDB for efficient data management, FastAPI for query processing and React to provide a user interface for visualizing and interacting with data. The API and web client are independent. The API could be used by other clients, such as a mobile application.

Database: MongoDB

MongoDB is the database of choice for efficiently storing and managing application data. Its flexible data model based on JSON documents makes it easy to store and retrieve information. Read/write operations are optimized for high performance.

API: FastAPI

FastAPI is used as a backend framework to manage HTTP requests and database operations. database operations. It offers high performance thanks to automatic documentation generation, rapid data rapid data validation and support for OpenAPI specifications.
